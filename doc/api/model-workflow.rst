Shared workflow steps
*********************

This module builds on the generic :mod:`.workflow` utilities/:class:`.Workflow` class
to provide steps that can be used for *any* workflow involving the MESSAGEix-GLOBIOM global model.

These are intended to be configured, adapted, or extended for new projects and applications,
instead of duplicate, project-specific code.
